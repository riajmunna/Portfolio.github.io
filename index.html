PROJECTS

  1. Customer Relationship Management (CRM)

    Customer relationship management is when a business or other organization administers its interactions with customers, typically using data analysis to study large amounts of information. This system was developed for the GOVT organization (SEIP-BACCO). Here super admin, customer, and agent can use this system. Only the super admin can add agents, customers, employees, and users can perform updates and delete them. The agent can do the same things for employees and users and lastly, employees can add update-only users, that's why we use middleware. For super admin here used JStream authentication and others for user-defined authentication. The main feature of this system is how much time and data entry done by employees is recorded here which can see only super admin and agents.


    Code: https://github.com/riajmunna/crm.git
